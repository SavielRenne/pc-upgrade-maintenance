# Project: PC Upgrade and Preventive Maintenance

---

## Objective

Upgrade a personal computer with the installation of an M.2 NVMe SSD and perform preventive maintenance to improve performance and ensure system longevity.

---

## Materials Used

- M.2 NVMe SSD (Kingston NV3)
- Precision screwdriver set
- Anti-static brush and cloth
- Isopropyl alcohol (optional)
- Thermal paste (if required)
- System migration/cloning software (Macrium Reflect / Acronis / Clonezilla)

---

## Step by Step

### 1.Preparation

- Shut down the PC and unplug it from the power source.
- Press the power button for a few seconds to discharge residual energy.
- Use an anti-static wrist strap or work on an anti-static surface.

---

### 2.Installing the M.2 NVMe SSD

- Locate the M.2 slot on the motherboard.
- Insert the SSD at approximately a 30° angle.
- Gently press it down and secure it with the retention screw.
- Double-check that the SSD is properly seated.

---

### 3.System Migration (Optional)

- Connect both the old storage drive and the new SSD.
- Use cloning software to migrate the operating system.
- Restart the PC and access BIOS/UEFI.
- Set the NVMe SSD as the primary boot device.

---

### 4.Preventive Maintenance

- Open the PC case carefully.
- Remove dust using an anti-static brush and cloth.
- Clean fans, heatsinks, and air filters.
- Reapply thermal paste on the CPU (if necessary).
- Organize cables to optimize airflow.

---

### 5.Post-Installation Testing

- Boot the PC and confirm the SSD is recognized in BIOS.
- Run benchmarks (CrystalDiskMark, ATTO, etc.) to measure performance.
- Monitor system temperatures and stability under normal usage.

---

## Results

- Significantly reduced system boot time.
- Higher read/write speeds for applications and files.
- Improved airflow after cleaning and cable management.
- More stable and responsive system.
