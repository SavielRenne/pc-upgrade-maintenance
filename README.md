## Project Description  
This project documents a complete PC preventive maintenance and hardware upgrade process.

Activities performed:
- Internal cleaning
- SSD M.2 NVMe installation
- System migration
- Cable management
- Performance testing
- SSD health monitoring

## Performance Summary  

- SSD read speed increased from 517.51 MB/s to 3715.33 MB/s
- System boot time reduced from 50 seconds to 20 seconds
- Slight increase in idle temperature (35°C → 38°C) due to M.2 slot positioning under GPU airflow path. Temperatures remain within safe operating range (max 48°C under load).
